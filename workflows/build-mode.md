# Build Mode Workflow
### Prompt Person™ AI Agent Starter Kit

---

## When to Use
When you want your agent to BUILD something for you — a website, app, script, automation, tool, or anything else you can describe.

## How It Works

### Step 1: Describe What You Want
Tell your agent what you want to build in plain English. You don't need to know how to code. Examples:
- "Build me a personal website"
- "Create a script that organizes my downloads folder"
- "Make a simple budget tracker"
- "Build a tool that renames all my files"

### Step 2: Agent Breaks It Down
Your agent will outline the steps, ask clarifying questions, and confirm the plan before building. It might ask:
- "Where do you want this saved?"
- "What should it look like?"
- "Any specific features?"

### Step 3: Build Together
Your agent builds it piece by piece, showing you progress as it goes. It creates real, working files on your computer. You can give feedback at each step:
- "Make the font bigger"
- "Add a contact form"
- "Change the color to blue"

### Step 4: Test & Iterate
Your agent will help you test what was built and fix anything that doesn't work. Keep refining until you're happy.

### Step 5: Save & Reuse
The finished project lives on your computer. You own it. You can modify it anytime by asking your agent to make changes.

## Important
Your agent can build real, functional things — not just text. It writes actual code, creates actual files, and builds actual tools. This is what makes an AI agent different from ChatGPT.
