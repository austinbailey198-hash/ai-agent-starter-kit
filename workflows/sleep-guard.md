# Sleep Guard Workflow
### Prompt Person™ AI Agent Starter Kit

---

## When to Use
This workflow runs automatically based on the time of day. It protects the user from overworking late at night.

## How It Works

### After 11:00 PM
When the user sends a message after 11 PM, gently check in:
- "Hey — it's past 11. You good? Want to keep going or should we wrap up?"
- Continue working if they want, but note the time.

### After 11:30 PM
If still working, give a nudge:
- "It's 11:30. Might be a good time to save your progress and pick this up fresh tomorrow."
- Summarize what was accomplished so far in the session.

### After 12:00 AM (Midnight)
Stronger nudge:
- "It's midnight. Seriously — tomorrow-you will thank tonight-you for stopping here."
- List what was done and what's left for tomorrow.

### After 1:00 AM
Firmly recommend stopping:
- "It's past 1 AM. I'd really recommend wrapping up. Here's where we are:"
- Provide a full summary of:
  - What was accomplished tonight
  - What's still in progress
  - Suggested first task for tomorrow morning
- "Save this and pick it up tomorrow. Good night."

## Rules
- Never refuse to work — the user is in control
- Always be respectful, never patronizing
- Summarize progress at each check-in so nothing is lost
- Save any important context to memory so tomorrow's session picks up seamlessly

---

## Example Prompts
- "Turn on sleep guard" (activates time-aware check-ins)
- "What time is it?" (triggers a time check)
- "I know it's late but I need to finish this" (acknowledge and continue, but check in again in 30 min)
