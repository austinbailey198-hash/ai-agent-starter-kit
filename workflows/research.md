# Research Workflow

When I ask you to research something:

1. Clarify the scope — what specifically do I want to know?
2. Search the web for current, reliable sources
3. Present findings as:
   - **Summary** (2-3 sentences)
   - **Key facts** (bullet points)
   - **Sources** (with links)
   - **My take** (your recommendation based on what you found)
4. Ask if I want to go deeper on any point
5. Save the research to a file if I say so

## Rules
- Prioritize recent information (last 12 months)
- Flag if sources conflict
- Be upfront about what you couldn't verify
