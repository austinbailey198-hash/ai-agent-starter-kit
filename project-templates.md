# Prompt Person™ — 5 Real-World Project Templates

Copy-paste these prompts to your agent and it will build each project for you. No coding required.

---

## Project 1: Personal Website

**What it is:** A clean, professional website with your name, bio, links, and contact info.

**Say this to your agent:**
"Build me a personal website. My name is [YOUR NAME]. I'm a [YOUR ROLE]. Include a short bio section, links to my social media ([LIST THEM]), a section about what I do, and a contact email link to [YOUR EMAIL]. Make it dark themed, modern, and mobile-friendly. Save it in a folder called my-website."

**What you'll get:** A working HTML/CSS website you can open in your browser and deploy anywhere.

---

## Project 2: Budget Tracker

**What it is:** A simple tool to track income, expenses, and see where your money goes.

**Say this to your agent:**
"Build me a simple budget tracker. I want to be able to add income and expenses with a date, category, and amount. Show me a running total of how much I've spent vs earned. Keep it simple — a single HTML page with JavaScript is fine. Save it in a folder called budget-tracker."

**What you'll get:** A working budget tracker you can open in your browser and use immediately.

---

## Project 3: Email Automation System

**What it is:** A set of pre-written email templates your agent manages and customizes for each situation.

**Say this to your agent:**
"Create an email automation system for me. I need templates for: cold outreach, follow-up (3-email sequence), thank you, meeting request, and introduction. For each template, include placeholders I can fill in like [NAME], [COMPANY], [TOPIC]. Save them all in a folder called email-templates. Also create a master list that shows when to use each one."

**What you'll get:** A folder of ready-to-use email templates with a guide for when to use each.

---

## Project 4: Resume / Portfolio Site

**What it is:** A professional online resume that you can share with employers or clients.

**Say this to your agent:**
"Build me an online resume site. My name is [YOUR NAME]. I'm a [YOUR ROLE] with experience in [YOUR SKILLS]. Include sections for: summary, work experience, skills, education, and a contact button. Make it look professional and modern. Use a dark theme with purple accents. Save it in a folder called my-resume-site."

**What you'll get:** A polished, shareable resume website.

---

## Project 5: Custom Tool / Automation

**What it is:** A script or tool that automates something repetitive in your life.

**Say this to your agent:**
"Build me a tool that [DESCRIBE WHAT YOU WANT]. For example:
- organizes my Downloads folder by file type
- renames photos by date
- converts a folder of text files into one combined document
- scrapes a list of items from a website
- generates a weekly summary from my notes folder

Make it easy to run and include instructions for how to use it."

**What you'll get:** A working automation script you can run anytime.
