YOUR SAFETY MATTERS TO US

  This kit was built with your privacy and
  security as a top priority. Here's exactly
  what it does and doesn't do.


═══════════════════════════════════════════════


WHAT THE KIT DOES:

  ✓ Creates a folder on your computer
    (~/my-ai-agent or ~/my-ai-agent-trial)

  ✓ Writes text files (CLAUDE.md, memory
    files, workflow templates)

  ✓ Checks if Node.js and Claude Code are
    installed — installs them if not (with
    your permission)

  ✓ Asks your name, role, and goals to
    personalize your agent


WHAT THE KIT DOES NOT DO:

  ✗ Does NOT collect your personal data
  ✗ Does NOT send information to any server
  ✗ Does NOT access your email, contacts,
    or files without your permission
  ✗ Does NOT install hidden software
  ✗ Does NOT run background processes
  ✗ Does NOT modify system settings
  ✗ Does NOT require admin/root access


═══════════════════════════════════════════════


100% TRANSPARENT CODE

  The setup script (setup.sh) is a plain text
  file. You can open it in any text editor and
  read every single line before running it.
  There is nothing hidden, obfuscated, or
  encrypted. What you see is what it does.


═══════════════════════════════════════════════


100% LOCAL & PRIVATE

  Everything runs on YOUR computer. Your files,
  conversations, and agent data never leave
  your machine.

  The only external connection is to Anthropic's
  API (claude.ai) when your agent needs to
  think. This is the same connection used by
  millions of Claude users worldwide.

  We have zero access to:
  - Your conversations with your agent
  - Your files or documents
  - Your API keys or credentials
  - Anything on your computer
